<Item type='methode' titre='règles de Bioche' searchDesc="intégrale, changement de variable">

Si $f$ est un quotient faisant intervenir des polynômes de fonctions trigonométriques, on peut, pour claculer $\integ{a}{b} f(t) dt$ :

$\cd$ Si $f(-t) = f(t) \v \forall t \in \defff{a}{b}$, poser le changement de variable $t = \cos(u)$

$\cd$ Si $f(\pi-t) = f(t) \v \forall t \in \defff{a}{b}$, poser le changement de variable $t = \sin(u)$

$\cd$ Si $f(\pi + t) = f(t) \v \forall t \in \defff{a}{b}$, poser le changement de variable $t = \tan(u)$

</Item>

<Remarque titre=''>
Si aucun critère n'est rempli, il reste encore la possibilité de poser $t = \tan(\dfrac{u}{2})$ et d'utiliser les formules de l'angle moitié.
</Remarque>
